Streamlining Sensor Registration for BIM Digital Twins

Data Visualization Platform for Digital Twin Integration

Digital Twin BIM Integration IoT Data Visualization
Year 2024
Toolkit React · Node.js · Autodesk SimpleViewer
Instructor Shang-Hsien Hsieh
Recognition ICCBEI 2023 Proceedings

Digital twin technology optimizes energy use, driving the transition to Net Zero Energy Buildings (NZEB). However, high costs and complexity make it challenging for facility managers to develop their own systems without skilled personnel. Our research proposes a streamlined platform using Autodesk Platform Services (APS) and Building Information Modeling (BIM). It simplifies digital twin creation by integrating sensors, spatial management, energy data, and updates, with a us er-friendly interface for efficient, expertise-free management.

The system architecture follows a typical three-tier structure, consisting of the Application Layer, Business Layer, and Data Layer, moving from the user interface to the core data management functions. Each layer contains specific functional modules to support the integrated management of Building Information Modeling (BIM) and Internet of Things (IoT) sensor data.

It provides three types of data access for different user roles. Building Model Management allows administrators to create projects, update model versions, edit sensor information, and issue authorization codes. Sensor Registration enables registrants to upload sensor data and link real-time or historical records to the database. Model and Sensor Visualization allows general users to view BIM models and real-time sensor data. This workflow simplifies data maintenance and supports the development of building digital twins.
